## Getting started

1. Clone this repo and enter!

   ```bash
   git clone https://github.com/kasia-js/react-gallery.git
   ```

2. Install dependencies.

   ```bash
   npm i
   ```

3. Start the app.

  ```bash
  npm start
  ```
